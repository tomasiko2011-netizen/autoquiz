# **App Name**: GeoWhiz

## Core Features:

- Flag Display: Display a flag of a country to the user.
- Country Quiz: Present four country name options for the displayed flag and record user's choice.
- Capital Quiz: Present four capital city options for the selected country and record user's choice.
- Score Calculation: Calculate the user's score based on correct country and capital answers.
- Result Display: Display the final score and percentage of correct answers at the end of the session.
- Country Data Storage: Store country name, capital, and flag image URL in a database for quick and easy use during the quizzes. Could be stored using posgreSQL, MySQL or MongoDB).

## Style Guidelines:

- Primary color: Vivid blue (#4285F4) to represent knowledge and exploration.
- Background color: Light blue (#E3F2FD) to maintain a calm and educational environment.
- Accent color: Orange (#FF9800) to highlight correct answers and important UI elements.
- Headline font: 'Poppins', a geometric sans-serif font for headlines.
- Body font: 'PT Sans', a humanist sans-serif font for body text.
- Use simple, recognizable icons for navigation and feedback (e.g., a globe for starting the quiz, a checkmark for correct answers).
- Ensure the app uses an adaptive layout to display properly across devices
